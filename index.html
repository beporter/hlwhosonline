HLWhosOnline

HL Who's Online is a PHP script package that allows live status display of a Half-Life server in a web page. The script requires a web server running PHP 4+ and the IP of a Half-Life (or mod) game server.

I don't have time to come up with a big fancy web page for this project yet, so I'm just going to point you to the SourceForge project page.

Visit HL Who's Online on SourceForge.net!

I hope to have a demo running on this site soon as well. Right now, check out my Counter-Strike clan's homepage: www.icabbit.com, which is running the beta version of this package.
